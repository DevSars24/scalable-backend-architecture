## 💡 Theme of Real-World Examples

All examples in this guide use a **real e-commerce + admin dashboard** context:

- 🛒 **Shopping Cart** — state, props, context
- 🔐 **Auth/Login flow** — protected routes, reducers, tokens
- 📊 **Admin Dashboard** — HOC, lazy loading, performance
- 📦 **Product listing** — lists, keys, pagination, virtualization
- 📝 **Checkout form** — controlled/uncontrolled, validation
- 🔔 **Notifications** — side effects, cleanup, subscriptions
